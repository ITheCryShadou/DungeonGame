# Chaos Dungeon

Standalone React + Phaser roguelike game.

## Project Layout

- `react-frontend/src/game/` - game code, systems, UI, balance, and save helpers.
- `react-frontend/src/assets/game/` - runtime game images and sprite assets.
- `react-frontend/public/` - static browser assets.
- `docs/` - game documentation.

## Run

```bash
cd react-frontend
npm install
npm run dev
```

## Build

```bash
cd react-frontend
npm run build
```
